# LabQC Example Data Sets

This folder contains realistic example data files for testing and demonstrating all modules of the LabQC application. The data covers **all major clinical laboratory disciplines** -- molecular diagnostics (SARS-CoV-2, HIV-1, HBV, HCV, TB/MTB), clinical chemistry, hematology, and immunoassay -- with values representative of real-world diagnostic laboratory operations.

Westgard rules are used across all clinical laboratory disciplines, not just molecular diagnostics. The QC principle is identical regardless of whether the measured value is a Ct value, a concentration, or a cell count: plot control values on Levey-Jennings charts and apply Westgard rules to detect systematic and random errors.

---

## How to Regenerate

Excel files are generated by two Python scripts:

```bash
cd /path/to/LabQC

# Molecular diagnostics (RT-PCR) examples
py -3.12 example_sets/generate_examples.py

# Clinical chemistry, hematology, immunoassay, and expanded sigma/validation
py -3.12 example_sets/generate_clinical_chemistry.py
```

Requires: `openpyxl` (install with `pip install openpyxl` if not already present).

Both scripts use a fixed random seed (`SEED = 42`) so results are reproducible.

## Validation Data (`validation_data/`)

**LabQC module:** Validation / Method Verification

### lod_sars_cov2.xlsx

**Study type:** Limit of Detection (LOD)

| Concentration | N Replicates | Expected Detection Rate | Expected Ct |
|---------------|-------------|------------------------|-------------|
| 0 (Blank) | 20 | 0% | Undetermined |
| 50 copies/mL | 20 | ~55% | ~37.0 |
| 100 copies/mL | 20 | ~80% | ~35.0 |
| 200 copies/mL (Claimed LOD) | 20 | ~95% | ~33.0 |
| 500 copies/mL | 20 | 100% | ~30.5 |
| 1000 copies/mL | 20 | 100% | ~28.5 |

Columns: Sample ID, Target, Concentration (copies/mL), Replicate, Ct Value, Detection (Pos/Neg), Run Date, Operator

**How to use:** Upload in the LOD study section of the Validation module. The application should calculate detection rates at each level and identify the concentration at which >= 95% detection is achieved.

### loq_hiv1.xlsx

**Study type:** Limit of Quantitation (LOQ)

Six concentration levels from 100 to 10,000 copies/mL, 10 replicates each. CV% decreases with increasing concentration (realistic behavior). Includes back-calculated concentrations from Ct values.

Columns: Sample ID, Target, Nominal Concentration, Log10 Nominal, Replicate, Ct Value, Measured Concentration, Log10 Measured, Run Date, Operator

**How to use:** Upload in the LOQ study section. The application should calculate CV% at each level and determine the lowest concentration where CV% meets the acceptance criterion (typically <= 20%).

### precision_intra_hbv.xlsx

**Study type:** Repeatability (intra-run precision)

20 replicates of L2 HBV DNA control in a single run. Expected Ct ~24.0 +/- 0.5.

**How to use:** Upload in the Precision section. The application should calculate mean, SD, and CV% from the 20 replicates and compare against acceptance criteria.

### precision_inter_hbv.xlsx

**Study type:** Reproducibility (inter-run precision)

5 runs over 5 consecutive days, 4 replicates per run. Includes run date and operator columns to enable variance component analysis.

**How to use:** Upload in the Precision section. Tests the inter-run (between-day) precision calculation including within-run and between-run variance components.

### linearity_hcv.xlsx

**Study type:** Linearity / Reportable Range

7 dilution levels (100 to 100,000 copies/mL), 3 replicates each. Expected R-squared ~0.998. Includes a "Regression Summary" sheet with slope, intercept, and efficiency.

Columns: Sample ID, Target, Nominal Concentration, Log10 Nominal, Replicate, Ct Value, Measured Concentration, Log10 Measured

**How to use:** Upload in the Linearity section. The application should perform linear regression on log10(concentration) vs Ct and report slope, intercept, R-squared, and efficiency.

### precision_glucose.xlsx

**Study type:** Intra-run and inter-run precision for Glucose (Clinical Chemistry)

Two sheets in one workbook:
- **Intra-Run Precision:** 20 replicates of L1 (Normal) glucose in a single run on Cobas c501. Mean = 95 mg/dL, SD = 2.5.
- **Inter-Run Precision:** 5 days, 4 replicates per day. Includes day-to-day variation.

Columns: Sample ID, Analyte, Level, Replicate, Value (mg/dL), Run Date, Instrument, Operator

**How to use:** Upload in the Precision section. Demonstrates precision evaluation for a chemistry analyte (concentration-based) rather than a molecular analyte (Ct-based).

### linearity_creatinine.xlsx

**Study type:** Linearity / Reportable Range for Creatinine (Clinical Chemistry)

7 concentration levels (0.5, 1.0, 2.0, 5.0, 10.0, 15.0, 20.0 mg/dL), 3 replicates per level. Includes a proportional bias of ~2% and realistic measurement noise.

Columns: Sample ID, Analyte, Nominal Concentration (mg/dL), Replicate, Measured Value (mg/dL)

**How to use:** Upload in the Linearity section. The application should perform linear regression on nominal vs measured values, report slope, intercept, R-squared, and per-level recovery. Acceptance: slope 0.95-1.05, R-sq >= 0.995.
